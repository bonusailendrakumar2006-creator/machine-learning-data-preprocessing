# machine-learning-data-preprocessing
Complete Machine Learning Data Preprocessing using Python, Pandas and Scikit-learn.

Machine Learning - Data Preprocessing

📌 Overview

This project demonstrates the complete Data Preprocessing pipeline used before training a Machine Learning model.

The dataset is cleaned, transformed, and prepared using Scikit-learn and Pandas.

---

 🚀 Topics Covered

- Handling Missing Values
- SimpleImputer
- Encoding Categorical Data
- One-Hot Encoding
- Label Encoding
- Train-Test Split
- Feature Scaling
- StandardScaler
- MinMaxScaler

---

 🛠 Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

---

📂 Dataset

Sample employee dataset containing:

- Country
- Age
- Salary
- Purchased

---

📖 What I Learned

During this project I learned:

- Why missing values must be handled before training.
- Why categorical data needs encoding.
- The difference between Label Encoding and One-Hot Encoding.
- Why datasets are split into training and testing sets.
- Why Feature Scaling is important and which algorithms require it.
- How to preprocess data using Scikit-learn.
